# ChemCrow2

Интеллектуальный агентный ассистент для химиков нового поколения, созданный командой хакатона **ITMO 2026**.

Проект объединяет современные LLM-технологии с инструментами хемоинформатики, предоставляя учёным и специалистам удобный интерфейс для решения сложных химических задач прямо в диалоге.

## Возможности

- **Поиск и анализ соединений** — получение информации по молекулам, SMILES, IUPAC-именам
- **Прогнозирование свойств** — оценка физико-химических характеристик молекул с помощью RDKit и ML-моделей
- **Планирование синтеза** — ретросинтетический анализ и построение маршрутов реакций
- **Генерация протоколов** — автоматическое составление экспериментальных процедур
- **Редактор молекул** — визуальный ввод структур через встроенный Ketcher
- **Мультиагентный диалог** — сложные задачи решаются цепочкой специализированных AI-агентов

## Стек технологий

| Слой | Технологии |
|------|-----------|
| **Backend** | FastAPI, Python, PostgreSQL, Redis, Celery |
| **AI / LLM** | LangChain, LangGraph, OpenAI GPT-4 |
| **Хемоинформатика** | RDKit, Ketcher |
| **Frontend** | React 19, TypeScript, TanStack Router, Tailwind CSS |
| **Инфраструктура** | Docker, Traefik |

## Архитектура

Монорепо с микросервисной структурой:

```
itmo_fatsapi/
├── backend/          # FastAPI + SQLModel + Alembic + Celery
├── frontend/         # React 19 + Vite + shadcn/ui
└── services/         # AI-агент, ретросинтез, MCP-инструменты (в разработке)
```
